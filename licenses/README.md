# License Notes

This folder stores human-readable rights notes for Bible and hymnbook sources.

These notes are not legal opinions. They are repository-level provenance and
review records that explain why a catalog entry is considered production-ready,
testing-only, or still pending review.

## Suggested Workflow

1. Find the upstream rights statement in the source file or on the official
   source page.
2. Copy the relevant details into a short note here.
3. Record the result in the catalog using:
   - `rightsStatus`
   - `allowDownload`
   - `allowBundling`
   - `license`
   - `licenseUrl`
   - `licenseNotesPath`
4. If the rights are unclear, keep the item in `catalog/research` or
   `catalog/testing`, not `catalog/production`.
